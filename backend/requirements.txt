# websockets.asyncio (used internally by supabase's `realtime` dependency,
# in realtime/_async/client.py) was only introduced in websockets 13.0.
# The previous pin here (>=11,<13) actively excludes every version that
# has that module, so a fresh install crashes at import time with:
#   ModuleNotFoundError: No module named 'websockets.asyncio'
# `realtime` is a hard transitive dependency of `supabase` (it's imported
# unconditionally in supabase/__init__.py, even though this project never
# uses Supabase's realtime subscriptions) — so the floor must be >=13, not
# capped below it. No upper bound: websockets has been stable here through
# 16.x and capping it again is what caused this exact bug.
websockets>=13
